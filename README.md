# 说明

导出百度指数**当前显示的数据**，导出格式为 CSV。

目前会导出两个 CSV 文件：
* 第一个 CSV 文件中包含**关键词的搜索指数**数据。
* 第二个 CSV 文件中包含时间范围内出现的**新闻头条**数据。

# CSV 文件说明

## 关键词的搜索指数 CSV 文件

文件名命名规则：`[关键词1,关键词2...]_[开始时间]-[结束时间].csv`

文件内容：

|      | 时间点1  | 时间点2  | 时间点3  | 时间点4  | 时间点5  | ... |
|------|-------|-------|-------|-------|-------|-----|
| 关键词1 | 搜索指数1 | 搜索指数2 | 搜索指数3 | 搜索指数4 | 搜索指数5 | ... |
| 关键词2 | 搜索指数1 | 搜索指数2 | 搜索指数3 | 搜索指数4 | 搜索指数5 | ... |
| ...  | ...   | ...   | ...   | ...   | ...   | ... |

## 新闻头条 CSV 文件

文件名命名规则：`[关键词1,关键词2...]_[开始时间]-[结束时间].news.csv`

文件内容：

|      | 时间点(与*关键词的搜索指数 CSV*中的时间点) | 新闻实际发布时间 | 新闻来源 | 新闻标题 | 新闻 url |
|------|---------------------------|----------|------|------|--------|
| 关键词1 | 时间点1                      | 发布时间1    | 来源1  | 标题1  | url1   |
| 关键词2 | 时间点1                      | 发布时间2    | 来源2  | 标题2  | url2   |
| ...  | ...                       | ...      | ...  | ...  | ...    |
